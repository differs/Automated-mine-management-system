## 八、使用建议

### 优先联系策略

```
第一批（本周打）：国家能源集团、中煤集团、山东能源、陕煤集团、山西焦煤
  → 他们有专门的数字化/智能化部门，有预算，政策驱动必须做
第二批（本月打）：紫金矿业、江西铜业、洛阳钼业、金川集团、华阳新材料
  → 大型金属矿山，运输调度痛点明显
第三批（持续打）：各省中型煤矿（如盘江煤电、淮北矿业、开滦集团等）
  → 中型矿山，性价比最高的客户群
```

### 对接部门建议

- **大型集团** → **数字化部/智能化部/生产调度中心**（他们负责数字化转型预算）
- **中型矿山** → **矿长/生产副矿长**（直接决策者）
- **小型矿山** → **老板/总经理**（一人决策）

### 企业名录批量采购推荐

> ⭐ **最省力的方案**：花 **￥650 买一套全国采矿企业名录 Excel**（百库信息），拿到后用 Python 按以下条件筛选：
> - 注册资本 ≥ 1000万（排除极小矿山）
> - 行业 = "煤炭开采"/"金属矿采选"
> - 省份 = 山西/内蒙古/陕西/贵州/河南/山东/安徽/新疆
> - 经营状态 = "存续/在业"
> 
> 预计可筛选出 **3,000~5,000家优质目标客户**，然后用邮件/电话批量触达。
