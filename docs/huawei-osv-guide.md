## 一、什么是矿鸿OSV

**OSV = Operating System Vendor（操作系统发行版厂商）**

矿鸿OSV伙伴是基于华为矿鸿操作系统，为矿山行业开发操作系统发行版、适配硬件设备、提供解决方案的合作伙伴。

**当前状态：**
- 已有 **29家** OSV伙伴（截至2026年4月）
- 计划3-5年突破 **500家** 生态伙伴
- 认证设备已达 **935款**
- 窗口期仍在，但名额在减少

## 二、申请流程（6步）

```
Step 1: 下载矿鸿OSV申请模板
   → https://www.mineharmony.cc/khrz/36996.jhtml (或联系华为获取)
  
Step 2: 填写申请信息
   → 企业能力、鸿蒙/HarmonyOS经验、商业机会、团队配置

Step 3: 发送至华为
   → 邮箱：mineharmony@huawei.com
   → 标题建议：「矿鸿OSV申请 - 企业名称」

Step 4: 华为内部评审
   → 评审维度：企业能力、鸿蒙经验、商业机会
   → 评审通过后获得临时DP平台和issue reporter权限

Step 5: 获取矿鸿套件 + 技术支持
   → 开始技术适配和开发

Step 6: 正式认证
   → 1年内达到评分≥60分 → 正式成为OSV伙伴
```

## 三、评审标准（要达到60分以上）

根据《2026版矿鸿OSV生态合作伙伴认证与管理策略》：

| 评分维度 | 说明 |
|:--------|:----|
| **企业技术能力** | 是否有软件开发团队、嵌入式开发经验 |
| **鸿蒙/HarmonyOS经验** | 是否有OpenHarmony或HarmonyOS开发经验 |
| **矿山行业积累** | 是否了解矿山业务，有矿山客户 |
| **商业机会** | 是否有明确的商业计划和落地场景 |
| **团队配置** | 是否有专职的鸿蒙开发团队 |

**我们的优势：**
- ✅ 有Rust/Flutter全栈开发能力
- ✅ 有完整的矿山调度产品
- ✅ 已有398家矿山企业数据，了解行业
- ✅ 产品可运行在矿鸿设备上（ARM/x86 Linux均可）
