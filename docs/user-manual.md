# 用户操作手册

## 角色说明

系统定义了 6 种后台角色 + 1 种司机身份：

| 角色 | 权限范围 | 常用操作 |
|------|---------|---------|
| **super_admin** | 系统全部权限 | 管理用户、配置系统 |
| **dispatcher** | 调度中心 | 派单、监控队列、处理异常 |
| **pit_operator** | 坑口作业 | 叫号、确认装车 |
| **weigh_operator** | 地磅操作 | 称重录入、异常上报 |
| **finance** | 财务结算 | 查看报表、导出数据 |
| **ops_analyst** | 运营分析 | 查看看板、效率分析 |
| **driver** | 司机（移动端） | 接收任务、到场打卡、排队 |

## 调度后台（Web）

### 1. 登录

1. 打开浏览器访问调度后台地址
2. 输入用户名和密码
3. 选择对应角色登录

### 2. 首页看板

登录后进入看板页面，显示：

- **今日概览**：派单数、完成数、在途数、取消数
- **实时队列**：各坑口当前排队数量和平均等待时间
- **完成吨位**：今日累计完成吨位
- **异常提醒**：待处理的告警列表
- **坑口效率**：各坑口的趟次和吞吐量排名

### 3. 司机管理

#### 查看司机列表
- 左侧导航点击"司机管理"
- 列表显示姓名、电话、车牌、车型、状态
- 支持按关键字搜索（姓名/电话/车牌）
- 支持按状态过滤（空闲/工作中/离线）

#### 添加司机
1. 点击"新增司机"按钮
2. 填写姓名、手机号、车牌号、车型、额定载重
3. 点击"保存"

#### 批量导入
1. 点击"导入"按钮
2. 下载导入模板（CSV 格式）
3. 按模板格式填写司机信息
4. 上传文件导入

### 4. 坑口管理

#### 查看坑口列表
- 左侧导航点击"坑口管理"
- 列表显示名称、编号、当前排队数、平均等待时间、状态

#### 管理坑口
- 新增/编辑坑口名称、编号、位置、排队容量
- 启用/停用坑口（停用后不再接受派单）

### 5. 运单管理

#### 创建运单
1. 点击"新建运单"
2. 选择坑口、选择司机
3. 输入预估重量（可选）
4. 点击"创建"

#### 派单
1. 在运单列表中找到"待派车"状态的运单
2. 点击"派单"按钮
3. 确认派单

#### 取消运单
1. 找到进行中的运单
2. 点击"取消"按钮
3. 填写取消原因
4. 确认取消

### 6. 队列监控

- 查看每个坑口的实时排队情况
- 显示排队序号、司机姓名、车牌、入队时间
- 标记异常（超时未到、插队等）

### 7. 告警中心

- 列表显示所有告警
- 按类型过滤（超时、重量偏差、离场未称重等）
- 处理告警：标记为"已处理"
- 告警详情包含关联运单和操作建议

### 8. 报表统计

- 日/周/月报表
- 趟次统计
- 吨位统计
- 坑口效率排名
- 司机效率排名
- 支持数据导出（Excel/CSV）

## 司机移动端（H5）

### 访问方式

1. **微信内打开**：扫描二维码或在微信内打开链接
2. **浏览器打开**：在手机浏览器中输入地址

### 功能

#### 登录
- 使用手机号 + 验证码登录
- 或使用微信一键登录

#### 首页
- 显示当前任务状态
- 有任务时：显示坑口名称、排队位置、预计等待时间
- 无任务时：显示"暂无任务"

#### 任务操作
1. **接收任务**：查看调度派发的运单
2. **到场打卡**：到达坑口后点击"我已到达"
3. **排队状态**：实时查看排队序号和前面人数
4. **叫号通知**：轮到时收到通知
5. **历史记录**：查看已完成趟次和累计吨位

#### 个人中心
- 查看个人信息
- 修改手机号
- 查看累计趟次/吨位

## 坑口操作端（H5）

### 访问方式
- 坑口管理员可通过手机浏览器或平板访问
- 建议使用大屏设备（平板或专用终端）

### 功能

#### 队列管理
- 查看当前坑口的所有排队车辆
- 按顺序显示：序号、车牌、司机姓名、入队时间
- 标记异常车辆（插队、超时等）

#### 叫号操作
1. 查看队列中第一位车辆
2. 点击"叫号"
3. 该司机收到通知
4. 司机开始装车

#### 装车确认
1. 司机装车完成后点击"完成装车"
2. 系统记录装车时长
3. 运单流转到称重环节

#### 历史记录
- 查看当日已完成装车的趟次
- 查看平均装车时长

## 地磅操作端（H5）

### 访问方式
- 通过手机浏览器或地磅电脑访问

### 功能

#### 称重录入
1. 等待车辆上磅
2. 选择或确认运单
3. 录入毛重、皮重（可选）、净重
4. 点击"确认称重"
5. 运单自动完成

#### 异常处理
- 重量偏差超过阈值时自动提示
- 支持复磅和重量更正
- 记录异常原因

## 常见操作场景

### 场景 1：标准调度流程

```
调度员创建运单 → 派单给司机
                         ↓
司机收到通知 → 前往坑口 → 到场打卡
                         ↓
系统自动入队 → 排队等待 → 收到叫号
                         ↓
坑口管理员开始装车 → 结束装车
                         ↓
司机前往地磅 → 录入重量 → 运单完成
```

### 场景 2：司机超时未到

```
调度员发现司机超时未到场
  → 在运单中点击"取消"
  → 填写取消原因
  → 重新派单给其他司机
```

### 场景 3：插队处理

```
坑口管理员发现插队行为
  → 在队列中标记异常
  → 调整排队顺序（需填写调整原因）
  → 系统记录操作日志
```

## 操作规范

### 必须记录的操作

以下操作系统会自动记录操作人、操作时间和原因，请确保操作时填写真实信息：

- 取消运单
- 手工调整排队顺序
- 重量更正
- 人工代操作（替司机到场、替坑口叫号）
- 启用/停用坑口

### 异常上报规范

发现以下情况请及时在系统中上报异常：

| 异常类型 | 上报方式 | 处理建议 |
|---------|---------|---------|
| 司机超时未到 | 自动检测 | 调度员取消或改派 |
| 插队 | 人工上报 | 管理员调整队列 |
| 装车超时 | 自动检测 | 通知坑口管理员 |
| 重量偏差异常 | 自动检测 | 复核重量，确认是否复磅 |
| 离场未称重 | 自动检测 | 联系司机返回称重 |

## 故障排除

### 无法登录
1. 检查账号密码是否正确
2. 确认账号未被禁用
3. 联系管理员重置密码

### 页面加载失败
1. 检查网络连接
2. 尝试刷新页面
3. 清除浏览器缓存
4. 联系技术支持

### 无法签到/入队
1. 确认已收到派单
2. 确认已在坑口范围内（现场签到需要 GPS）
3. 联系调度员或坑口管理员代操作

### 称重数据异常
1. 确认车辆已完全上磅
2. 检查地磅是否归零
3. 如确认数据有误，使用"更正"功能
4. 记录更正原因
