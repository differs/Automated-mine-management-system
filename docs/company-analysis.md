## 1. 华为矿山军团 ❌ 不是客户，是生态合作方

**华为矿山军团的定位：**
华为不是矿山运营商，而是**为矿山企业提供ICT基础设施和AI平台的技术供应商**。

**核心产品：**
| 产品 | 说明 |
|:----|:----|
| **矿鸿操作系统** | 矿山设备互联互通，已认证935款设备，200+生态伙伴 |
| **矿山AI大模型** | 100+种AI算法（安全监测、设备巡检等） |
| **5G/工业承载网** | 井下/露天矿网络基础设施 |
| **无人驾驶方案** | 矿卡自动驾驶（与云鼎科技、徐工合作） |
| **智能冶炼方案** | AI驱动的冶炼优化 |

**对我们的意义：**
```
不是客户，而是生态合作伙伴。

合作方式：
  我们的调度系统 → 运行在华为云/矿鸿平台上
  我们的数据 → 对接华为AI大模型做智能调度
  模式 → 成为华为矿山生态伙伴，借华为渠道卖产品
  
对接联系人：华为矿山军团合作伙伴部门
```

## 2. 万丰集团 ⚠️ 不是矿企，暂不联系

**实际业务：**
万丰奥特控股集团的核心业务是：
- **汽车部件**（铝轮毂、镁合金）— 全球领先
- **航空工业**（通航飞机制造）
- **智能装备**

他们在招「资深调度员」是**制造业内部物流调度**，不是矿山运输调度。

**结论：**
```
❌ 不是目标客户
❌ 不需要矿山运输调度系统
✅ 跳过，不联系
```

## 3. 寒锐钴业 ✅ 最值得联系的客户

**基本信息：**
```
上市代码: 300618.SZ
主营业务: 钴、镍、铜矿开采与冶炼
海外矿山: 刚果(金)钴矿 + 印尼镍项目（在建，投资数十亿）
招聘信号: 「现场车辆调度员(外派印尼)」13-15k·13薪
```

**为什么是需要我们的客户：**

| 信号 | 含义 |
|:----|:----|
| 🚨 **在招调度员** | 矿区还在靠人调度，没有系统 |
| 🚨 **海外项目在建** | 印尼4万吨电积镍项目正在建设，**新矿需要新系统** |
| 🚨 **刚果金矿区运营中** | 现有矿区需要管理优化 |
| ✅ **上市公司** | 有预算，付款能力强 |
| ✅ **已上市20年** | 成熟企业，不是初创公司 |

**痛点场景：**
```
刚果(金)矿区：
  - 钴矿运输调度，多坑口、长距离
  - 当地网络条件差 → 需要离线调度
  - 中方管理人员远程管理 → 需要实时看板

印尼新项目：
  - 新建矿山，系统从零开始
  - 镍矿运输调度
  - 跨语言（中文/印尼语/英语）
```

**建议行动：**
```
优先级: ⭐⭐⭐（最高）
联系方式: 需找HR邮箱
跟进话术:
  "看到贵公司在招印尼矿区的现场车辆调度员，
  我们做矿山运输调度系统，支持离线可用，
  适合海外矿区网络不稳定的环境。"
```
